Calculadora em HTML e CSS

<p>🧾 Explicação do Código – Calculadora básica

Como estudante do primeiro semestre de Engenharia de Software, desenvolvi este projeto simples em HTML e CSS para treinar e entender melhor os fundamentos na prática.

O programa simula uma calculadora onde o usuário informa os números que deseja realizar suas operações e então o sistema responde.

Além da parte técnica, busquei alinhar o tema com minha meta profissional de atuar como DEV no futuro, aproximando desde cedo meus estudos da realidade de projetos funcionais. Essa experiência também serviu para praticar versionamento com Git/GitHub, seguindo as boas práticas da minha matéria de Bootcamp na faculdade. </p>
